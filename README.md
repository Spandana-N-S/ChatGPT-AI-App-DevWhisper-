# DevWhisper – AI Coding Assistant (ChatGPT-style)


Elegant conversational AI for developers: explain code, translate languages, suggest projects, solve challenges, clean chat UI.

## Features
- Real-time chat with GPT-powered responses
- Code explanation, translation, debugging help
- Project idea brainstorming
- Typing effect & loading animations

## Tech Stack
- Frontend: Vite + Vanilla JS/HTML/CSS
- Backend: Node.js + Express
- AI: OpenAI API (GPT-3/4)
- Deploy: Render (backend) + Vercel (frontend)

## Quick Start
1. `git clone ...`
2. `npm install`
3. Add OpenAI API key in .env
4. `npm run dev`
